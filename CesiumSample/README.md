# cesium-plugin-x-sample

用于构建Cesium扩展的各类插件、组件和相关示例模板

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

 启动 start

```sh
npm run start
```



## Running

```sh
node dist/bundle.js
```
